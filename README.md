## 功能

> 注意：本应用只适用于清华大学学生

### 基础功能

- 第一时间看到老师最新发布的通知及其附件
- 预览或下载课程文件，将文件分享到其他应用
- 一览作业要求，不错过任何 deadline，实时查询作业提交与批改情况

### 高级功能

- 课表日历同步
- Deadline 日历同步
- 归档内容
- 提交作业
- 收藏内容
- 隐藏课程
- 暗黑模式
- 全局搜索
- 切换学期
